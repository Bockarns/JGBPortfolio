# Jerome Gordon-Bock Portfolio

A personal portfolio website showcasing my projects, skills, and journey as a web development student at TUC Sweden.

**Live Site:** https://bockarns.github.io/JGBPortfolio/

## About

This is my personal portfolio site built with HTML, CSS, and JavaScript. It demonstrates my work and skills to potential employers and collaborators.

## Features

- **Home Section** - Introduction with animated name typing effect
- **About Section** - Education background at TUC Sweden with course progress
- **Skills Section** - Frontend, Backend, and Tools expertise
- **Projects Section** - Showcase of completed projects
- **Contact Section** - Contact form and social links

## Tech Stack

- HTML5
- CSS3
- JavaScript
- No frameworks - pure vanilla web technologies

## Projects Included

- WebLaunch PB - Web development service site
- Golfklubb Centar - Golf club website
- Personal Site - First portfolio website
- Dice Game - Multiplayer dice rolling game
- Guess My Number - Number guessing game with highscore
- Memory Game - Timed card matching game

## Setup

Simply open `index.html` in a web browser. No build steps or dependencies required.
